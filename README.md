損保一般資格勉強で時間がなかったため、スプレッドシートに間違えた問題をコピペして貼り付けるという力技で突破した。
その辛酸をなめた経験から誕生したアプリ。


良い点
・キーボードを一切使わない
->マウスを持ち続けられる、キーボードをチラ見せず画面を見続けられる
・難しい操作不要
->PC初心者でもマウスでドラッグすることぐらい知っている。その基本中の基本操作のみ
・アプリを使う際の脳死度が高い

悪い点
・現在改築中なため、車で例えるとエンジンと運転席とタイヤしかない。
->もちろん外枠のフレームやら、後部座席やら、カーナビやら、トランクルームやら、エアコンやら、ドアやら、シートベルト....が無い

自分がやりたかったことは表現できたので、あとは補助機能追加とコードを見やすく整理する。(死にコード撲滅、メソッドに色付け、拡張性を考える)
